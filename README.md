# NanoClaw 🧠

**Persistent AI personal assistant — runs 24/7 on your Android phone via Termux.**

NanoClaw is a mobile-first web app + CLI daemon that keeps your tasks, revenue opportunities, and AI conversations in one place. It connects to WhatsApp Business and Gmail, so you can interact with it from anywhere — your phone, a web browser, or a terminal.

---

## Features

### AI Chat Assistant
- Powered by **GPT-4o** (OpenAI) with full conversation memory per session
- Three specialized skill modules that auto-activate based on what you're asking:
  - **📋 Organization** — task capture, GTD-style prioritization, follow-up tracking
  - **💻 Coding** — architecture review, bug diagnosis, code generation, automation scripts
  - **💰 Entrepreneurship** — opportunity identification, revenue modeling, MVP planning, execution roadmaps
- Quick-start prompts on first load ("What should I focus on today?", "Find me a revenue opportunity", etc.)
- Auto-detects and creates tasks/opportunities/insights from your conversation — no manual entry needed
- Works without an API key (demo mode with pre-built responses)

### Task Manager
- Full CRUD: add, view, and complete tasks
- Priorities: `low` / `medium` / `high` / `critical` with color-coded borders
- Categories: Org, Coding, Entrepreneurship, Personal
- Filter tabs: All / Pending / In Progress / Done with live counts
- Tasks can be created manually or auto-generated by the AI from chat

### Opportunity Tracker
- Pipeline dashboard: total count, active opportunities, and total estimated revenue
- Opportunity types: side hustle, investment, partnership, client, product, service, automation
- Stage progression: Identified → Researching → Planning → Executing → Scaling → Completed
- Tap any opportunity to view details and advance its stage
- Estimated revenue and time-to-revenue tracking
- Auto-logged when AI detects a revenue opportunity in chat

### Proactive Insights
- AI generates insights automatically: opportunity alerts, task reminders, revenue tips, code suggestions
- Daily brief generator with pending tasks, active opportunities, and unread insights
- Insights are tracked as read/unread

### Integrations
- **WhatsApp Business** — receive and respond to messages via Meta's Cloud API
- **Gmail** — read and process emails (andycodebot@gmail.com)
- **Web interface** — mobile-first dark UI, optimized for phone use
- **Termux CLI** — run `nanoclaw "your question"` directly in your terminal

### Termux / Android Setup
- Install script for Termux on Android (S24 Ultra or any Android device)
- Auto-starts on every reboot via Termux:Boot
- Persistent daemon runs in the background

## Getting Started

### Prerequisites

- [Bun](https://bun.sh) installed
- Node.js 20+
- OpenAI API key (optional — works in demo mode without it)

### Installation

```bash
# Clone the repo
git clone <your-repo-url>
cd nanoclaw

# Install dependencies
bun install

# Run database migrations
bun run db:migrate

# Start the dev server
bun dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Environment Variables

Create a `.env.local` file in the project root:

```env
# Required for full AI responses
OPENAI_API_KEY=sk-...

# Optional: use a different model (default: gpt-4o)
OPENAI_MODEL=gpt-4o

# WhatsApp Business (Meta Cloud API)
WHATSAPP_ACCESS_TOKEN=...
WHATSAPP_PHONE_NUMBER_ID=...
WHATSAPP_MY_PHONE_NUMBER=+1234567890

# Gmail integration
GMAIL_ACCESS_TOKEN=...
GMAIL_REFRESH_TOKEN=...
```

## Termux Setup (Android)

Run NanoClaw persistently on your Android phone:

1. Install **Termux** from [F-Droid](https://f-droid.org/packages/com.termux/) (not Play Store)
2. Install **Termux:Boot** from F-Droid
3. Open Termux and run:

```bash
curl -fsSL https://your-app-url.kiloapps.io/scripts/install-nanoclaw.sh | bash
```

4. Follow the setup prompts
5. The CLI starts automatically on every reboot

**Usage in Termux:**
```bash
nanoclaw "what should I focus on today?"
nanoclaw "find me a revenue opportunity"
nanoclaw "help me debug this error"
```

## WhatsApp Setup

1. Go to [developers.facebook.com](https://developers.facebook.com) → My Apps → Create App
2. Select "Other" → "Business" → Create
3. Add WhatsApp product to your app
4. Get your **Phone Number ID** and **Access Token**
5. Add them as environment variables (`WHATSAPP_PHONE_NUMBER_ID`, `WHATSAPP_ACCESS_TOKEN`)
6. Configure the webhook in the Meta console:
   - **Webhook URL:** `https://your-app.kiloapps.io/api/telegram`
   - **Verify Token:** `nanoclaw_verify_token`
